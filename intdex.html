COCOMO SIMULATOR

A application program for estimating software development


INTRODUCTION

The COCOMO Simulator is an application program that is used for compute estimation cost like effort time for developement and number of staff required of a software to be developed, the estimation are done by some constraints from user and some fixed constant provided for each type of cocomo model loke basic , semi-detached and detailed cocomo. And for calculation of effort adjustmant factor we need constraint from user in various condition for computing estimation.
